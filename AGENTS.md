## Project Summary
Winstone Castle is a luxury real estate landing page for a premium residential project in Hebbal, North Bangalore. The design is exclusive, modern, and iconic, focusing on visual storytelling and high-end conversion.

## Tech Stack
- Framework: Next.js 15 (App Router)
- Styling: Tailwind CSS
- Animation: Framer Motion
- Fonts: Playfair Display (Serif), Plus Jakarta Sans (Sans)
- Icons: Lucide React

## Architecture
- `src/components/landing/`: Contains all modular landing page sections.
- `src/app/page.tsx`: Main entry point assembling the sections.
- `src/app/globals.css`: Luxury theme configuration with oklch colors.

## User Preferences
- Senior UI/UX designer aesthetic
- Conversion-focused copy
- Non-traditional section order
- Asymmetric and layered sections
- Luxury tone (Refined, Aspirational)

## Project Guidelines
- Follow the Winstone Castle brochure theme.
- Focus on "Pure Living" and "Discerning Few" narratives.
- Use sensory language for living spaces.
- Present amenities as experiences, not lists.

## Common Patterns
- Subtle scroll reveals using Framer Motion.
- Dark theme with gold accents for prestige.
- Minimalist navigation and clear CTA.
